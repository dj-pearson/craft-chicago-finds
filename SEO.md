SEO Strategy for Craft Local Marketplace
1. Technical Foundation (Launch-Ready)

Site Architecture:

Flat structure with all key vendor/product pages no more than 3 clicks from home.

Clear hubs: /vendors/, /categories/, /locations/, /guides/.

Each vendor gets a dedicated page with unique content.

Performance (Core Web Vitals 2025 standards):

LCP < 2.5s, INP < 200ms, CLS < 0.1.

Use WebP images, lazy loading, deferred scripts.

Mobile-First:

Touch-friendly filters (by location, craft type, price).

Sticky cart + checkout buttons.

Crawlability:

XML sitemap auto-updating with new vendors/products.

Robots.txt tuned for indexable product/vendor pages only.

Canonical tags on paginated and filtered URLs.
